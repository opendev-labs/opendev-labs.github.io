# Changelog

All notable changes to the **opendev-labs** ecosystem will be documented in this file.

## [1.2.0] - 2026-01-30

### Added
- **Office Hub Gateway**: New central entry point at `/office` for all OpenDev-Labs infrastructure.
- **OfficeDashboard Component**: Terminal-inspired management interface with cinematic entry protocols.
- **ProductPageTemplate**: Standardized template for high-fidelity product showcase pages.

### Fixed
- **VOID Navigation Protocol**: Overhauled `safeNavigate` to use standard SPA history management, resolving the "Initialize Node" button failure.
- **Route Prefixes**: Standardized `/void/*` routing to prevent route leakage and improve reliability.
- **Firebase State Drift**: Resolved authentication state synchronization issues between local sessions and Firestore nodes.

### Changed
- **Absolute Sharp-Edge Aesthetic**: 100% removal of rounded corners across the entire ecosystem. Every button, card, and status indicator now uses a strict 0% rounding protocol.
- **AuthPage Redesign**: Cinematic authentication flow with animated encrypted fragment protocols.
- **Header Professionalization**: Unified navigation with "Mission Control" gateway and sharp-edge mobile interface.
- **Iconography**: Standardized on square/sharp-edge framework icons for a professional industrial look.

## [1.1.0] - 2026-01-27
- **Initial VOID Integration**: Migration of the sovereign deployment platform.
- **Identity Foundation**: Establishment of the Sovereign ID Protocol.
- **Aesthetic Alpha**: Introduction of the high-fidelity dark mode theme.
